Grid Archives 0.8.0

Posted on 2010-12-09 | Edited on 2018-11-29 | In PHP , Share , Technique , WordPress

Grid Archives 0.8.0 finally support user customized css. People complained about losing css changes after plugin update. It won’t be a problem any more since all you configurations won’t be lost when you update the plugin. One user case is simply copying all the default Grid Archives css (you can find them from “Plugin Editor”) to the “Custom CSS Styles” textarea and made any modifications you like, save the changes and that’s it. You don’t need to worry they will be lost during plugin update any more. [ChangeLog]

  • added an option to allow user to custom the display stylesheets, these customizations won’t be lost after plugin update.
