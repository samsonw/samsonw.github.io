Update 51: 急性胃肠炎

Posted on 2011-01-13 | Edited on 2018-11-29 | In Life
Symbols count in article: 534 | Reading time ≈ 1 mins.

1. 上周（从周四到周日）规律之外的饮食与饭局以及对自己吃辣能力提升的过分高估，最终导致了这一周的得不偿失，原本的好心情随着接连不断的拉肚子也一齐被马桶冲掉 ((坐在马桶上，倒是消化掉了不少电子书… 所以倒也不是一无是处))，不知道是肠胃炎本身就是伴着感冒还是坐马桶坐多了着凉了，感冒咳嗽的接踵而至似乎显得相当水到渠成。 躺床上喝了好几天米粥的我，这几天脑子里面全是各种大鱼大肉，愚以为这是我肠胃即将恢复的前兆，内牛满面的期待中； 2. 烧饭说：”啥叫孤独，孤独不是在 BLOG 上写出来的，而是一个人回到租的房子里面打开笔记本只是为了放 Friends 制造点声音时候流露出来的。” 我顿时很感动，几年来少有的感动…… 啥叫感到呀，感动也不是在 BLOG 上写出来的，而是一个人回到租的房子里面打开笔记本只是为了重温烧饭 twitter 上对于孤独的定义而顿时流露出来的； 3. 欠着一堆事情，果然最近惰性抬头，愚坚持认为这和寒冷的天气不无关系；空调暖气解决了体表的寒冷，却对内心的寒冷依然无能为力，而这 —— 需要的是乐观、激情，哦，还有情怀…… 4. 生病中，不宜废话；废话伤神，且不低碳；故该吃肉的继续吃肉，该喝粥的继续喝粥，只要心情愉快就好，嗯，愉快就好～

Update 50: 2011

Posted on 2011-01-04 | Edited on 2018-11-29 | In Life
Symbols count in article: 792 | Reading time ≈ 1 mins.

1. blog空间商开始催着要我renew，还没拿定主意是换还是留，这事目前还得再议，是后话；只是觉得这2年说过去就过去了，一句话——太快；发现所有写blog的几乎都很喜欢年终总结，所谓承前启后，继往开来；我向来没有这种习惯，不忍看到自己那一年怎么又是碌碌无为，mlgb好失败，不得不避实就虚，于是每年年末都会郁闷的发现那一年的收获就是没有收获，总结便是没有总结；想做的没做成，不想做的自然也不会做成；这…也是一种情怀； 2. 钱老的事情这几天一直在关注，因为事情就发生在我们浙江，据说温州的现任某官员亦曾是从老家嘉兴调往，所以更增添了一份亲切与恐怖，李承鹏俨然相当的给力 ((据说2010最给力的网络词汇便是给力))，我连续几个晚上都是在刷着看他blog底下的评论，非常给力，呵呵；证据不足、事态尚不明朗的情况下便定棺定论的确稍显不妥，且不妥的有点可疑，无论是前面的公权利部门还是之后进入的观察团 ((这就像平时我们聊天，我并没有怀疑你，甚至都没质问你原因，你就开始自我解释起来一样))，虽然我一直很愿意相信这真的只是一起普交，不过请给我一点相信的理由先…… 而不是未卜便能先知的所谓的结论； 3. 分享和炫耀有时候只是一步之差，很多人举手投足（包括微博、签名等等）没有丝毫share的感觉，分明是种show off；也许他们只是内心空虚，也许他们只是为了show off而show off，我不知道，但我确实很理解；马斯洛说现代人类有五层需求，那也许就是第三抑或第四层需求吧，呵呵，怎么搞的跟inception的梦境层次一般…… 4. 2010里不开心的事情太多，很多事情不可抗拒，不以人的意志为转移，有些不快现在依然存在，只要一回家便能真切体会；2011，希望能找到新的奋斗目标，如果说新一年有什么目标的话，这就算一个…… 汗 5. 世上只有妈妈好，所以要听妈妈的话，呵呵呵～

Update 48: 梦想就是梦里面才能想的

Symbols count in article: 829 | Reading time ≈ 1 mins.

1. 标题是我刚刚看完 Inception 后的潜意识里面的第一反应，可能和我在现实抑或梦境旧已存在的 projection（还是应该叫影像，残留？） 有关…… 之前没看任何影评剧透的我也早已耳濡目染知道剧情有所谓的嵌套迭代（剧透如潮水啊，不对，洪水啊…），真的看了一遍后发现原来还有 exception （limbo还是迷失域？），说不定还有 garbage collection，俨然一部软件工程教材……；当然所谓仁者见仁，淫者见淫，比如坊间还流传着的什么数理逻辑解析（就是淫者…）… ok，咱们还是就此打住，我决定有空再看一遍（虽然我觉得蛮傻的），并非出于对坊间流传着的所谓争论感兴趣，只为第一遍所没有注意到的一些如今众所皆知的细节，譬如戒指。 2. 大学时代我的一位神秘天蝎好友曾向我力荐过海上钢琴师，我当时允诺过一定会看，上周五晚11点半我终于开始兑现诺言…（我向来言而有信，虽然后来我一不小心睡着了… 第二天起来后再补完…）里面的男配角我想应该练过斗鸡眼之类的绝活吧，只要一有特写，就会看到其在试图让其眼珠子左右不停晃动，每次都让我颇为其捏一把汗；男主角则是现在 lie to me 的主角，我猜想可能就是因为海上钢琴师里面对能够洞穿人心理的犀利的眼神的出色诠释，lie to me 才选中了其继续表演犀利眼神，呵呵，everything is connected～ 3. 上面提到 lie to me，突然发现我已经快大半年没看美剧了，24 大结局（貌似今年4，5月份的样子）后我还真没碰过任何美剧，不过想想貌似也没啥好遗憾的；伴随着某些激情的慢慢消退，很多激情和热情俨然早已荡然无存，也不知道是不是种悲哀… 说不定我应该感到欣慰才对～ 4. 上周一直有想法要给 douban-collections 加类似豆瓣的翻页功能，我已尽力努力以及费力的忍到了现在，不知道还能忍多久，也许梦里面我还真是个有情怀的人，呵呵，当然只限于梦境，没有嵌套……

Grid Archives 0.8.0

Posted on 2010-12-09 | Edited on 2018-11-29 | In PHP , Share , Technique , WordPress
Symbols count in article: 658 | Reading time ≈ 1 mins.

Grid Archives 0.8.0 finally support user customized css. People complained about losing css changes after plugin update. It won’t be a problem any more since all you configurations won’t be lost when you update the plugin. One user case is simply copying all the default Grid Archives css (you can find them from “Plugin Editor”) to the “Custom CSS Styles” textarea and made any modifications you like, save the changes and that’s it. You don’t need to worry they will be lost during plugin update any more. [ChangeLog]

  • added an option to allow user to custom the display stylesheets, these customizations won’t be lost after plugin update.
