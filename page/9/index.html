Update 66: 屌丝很忙

Posted on 2012-04-16 | Edited on 2018-11-29 | In Life , Share

1. 看人挑担不吃力，自己挑担嘴要歪，据说更加common一点的说法是自己挑担步步歇，但是我从小学到的就是嘴要歪的这个版本，且我觉得其更符合我这里想要表达的context；当然了，这里非指桑骂槐，声东击西，亦非顾左右而言他，而只是想陈述某个事实上的事实而已，屌丝如我基本上也没有看人挑担的福分，因为那会让人有一份莫名的罪恶感，于是那扁担几乎显而易见的即便看上去不像是啥美食佳肴，屌丝也总是想着帮忙分去一杯羹，虽然那听着好像还是去分蛋糕的一样的兴高采烈…… 2. 于是屌丝就牙疼了，牙疼虽不是病，但那并不意味着就没有药；纵使对于缓解此类不是病的病的药物始终心存怀疑，无奈疼痛难耐时屌丝最终承认还是需要来那么点心理暗示… 当时竟然也能奏效，只觉药到病除神奇不已，于是屌丝顿时释然，在接下去的两周里像吃饭一样吃完了那盒神药；两周后回家，屌丝表达了想要继续买两盒神药带走的愿望，不料被母亲训斥此类止痛麻药还是少吃为妙，上回几次还叮嘱不到万不得已不要开封，搞的跟锦囊妙计似的，只是屌丝俨然已经丝毫没有任何印象…… 3. 于是屌丝就又失去了心理暗示，又时不时的开始牙疼了，这也许就是传说中所谓的痛苦的轮回，屌丝的悲哀； 4. 屌丝还很阴谋论，一直坚信这世上先有阴谋（此处省略若干字），才有的阴谋论；不愤世嫉俗，却又时常被人误解为愤青；信奉彪悍的人生不需要解释，却又时不时的因为还不够彪悍而各种解释…… 5. 屌丝很忙，忙在不言中。

Update 65: 塞翁“失”马

Posted on 2012-03-21 | Edited on 2018-11-29 | In Life , Share

1. 晚上一饭局，骑车欣然前往，由于对国人素质提高有着过分的信心，遂不曾过分关注停车的方位以及姿势，于是悲剧就发生了…… 故事或者剧本本该到此结束，我应该点下六点省略号或者明确表示此处省略若干字，以留给读者想象的空间抑或演员自由发挥的空间；可惜故事本身就略有点戏剧性，所以我尚不能华丽丽就此嘎然而止，于是我做出了一个艰难的决定，换到channel 2…… cut—— 2. 这不是倒叙，所以咱们还是继续走时间轴：前面说到未曾过分关注地理方位，说明其实我还是略微的那么关注了一下下，只不过正巧赶着时间又凭着不知道从哪里来的对该区域熟悉程度的自信，于是就没有抓住不放（又不是政治斗争，干嘛非要抓住不放呢，真是的……）；不过这份哪怕只有1%的不放心在我走回来的路上就被无限的放大了，我开始关注到这一带貌似好像确实没有我想象的那么安全，这可能可以归功于我的阴谋论心理，也可能归功于某种所谓的第六感（虽然我并没有小宇宙）； 3. 终于回到了停车的区域，我一扫视一警觉，车不在了……；再一扫视回来一放心，车好像又在，只不过换了个姿势，并且斜靠在旁边那辆助动车上，宛如一对好基友，带你去看流星雨什么的，以至于我都不好意思上前打断； 4. 终于来到了车前（小样尼玛真够啰嗦的……），本以为是流星雨伴着大风导演了这一切，但谁能料到这锁还真的被人动过（你敢信吗），锁链上有明显痕迹，且锁孔已经略微错位，我矫正了半天才得以勉强插进钥匙；看到此情此景鄙人的第一反应便是四处张望，因为小偷打不定就在仓惶逃窜中（当然他也可能正神情自若的注视着我），就在我觉得我是身带正义感的到处环视时，我并没有觉察到这其实也是相当可疑的举止，因为路人们可不知道这车必定就是谁的，你这一环视，在你眼里是嫉恶如仇，也许在他们眼里那叫做贼心虚…… 幸好是朋友陪着我一起来取的车，所以我们最后即便再可疑也不至于百口莫辩，否则这事就真的太戏剧化了； 5. 骑回来的路上，通过回想、联想以及部分艺术加工，偷车贼撬锁未遂后气急败坏摔车走人的情景便一幕幕的浮现脑海，至于他为啥那么快或者不那么快的放弃，这就留给了我们永远的遐想空间； 后记，经此一劫，凶兆未卜，一说大难不死，必有那啥，那到底有啥呀；一说螳螂捕蝉，不知道什么在后，啊还有啥呀；再一说便是塞翁失马，焉知非福，不过话说我也好像没失嘛……

Vim-Task 1.1

Posted on 2012-03-06 | Edited on 2018-11-29 | In Share , Technique , Tools , Vim

Thanks to tUrG0n, vim-task won’t suffer from the pathogen issue anymore from now on, and we also have a new feature which will prepend a “-“ for a non-task line when toggling the task status. So by toggling the task status, we’re now actually toggling between 3 statuses of the task:

  • blank: normal text
  • pending: -
  • completed: ✓

all with the same 1 keyboard shortcut as before, as always. So neat! PS: Vim native regex ((to make the script portable, substitute() method will always use the ‘magic’ option in the regex match)) is pretty awkward to work with, unless you use some additional features ((but we don’t want more dependencies though…)), the regex will quickly become a mess, which maybe makes the person who reads your code thinks “wow, how complicated the regex is!”, but actually the truth is always: “well, it’s not, it’s just that ugly, man…”
