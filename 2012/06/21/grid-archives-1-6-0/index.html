Grid Archives 1.6.0

Posted on 2012-06-21 | Edited on 2018-11-29 | In PHP , Share , Technique , WordPress

As requested, this major release mainly address one issue/feature that grid archives should actually support in the very beginning, the sort direction option. Before 1.6.0, all the posts are sorted in descending order by default and there seems to be no easy way to change this… but that pain will be history now, starting from now :) To be compatible with previous versions, the default sort direction will remain as ‘desc’, so if you still want the month/post to be displayed in the descending order, as you wish.. you’re required to do nothing~ yay! [ChangeLog]

  • added an option to change the post sort direction (asec or desc)
  • make the plugin compatible with wordpress 3.4

As always, please visit the plugin home page for more details and click here to check out the live demo.
